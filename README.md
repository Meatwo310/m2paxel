# m2paxel

[GitHub](https://github.com/Meatwo310/m2paxel) | [CurseForge](https://www.curseforge.com/minecraft/mc-mods/m2paxel) | [Modrinth](https://modrinth.com/mod/m2paxel)

A Mod that adds "Paxel", a multi-tool combining Sword, Pickaxe, Shovel, and Axe functionalities.

## English

### Features
- 🔨 Can mine any block
- 💪 4 times the durability of regular tools
- 🪓 Attack power equivalent to an axe
- ⚔️ Attack speed equivalent to a sword, with sweeping attacks
- (Not implemented yet) Right-click behavior

### Recipe
Place a sword, pickaxe, shovel, and axe of the same material in the crafting grid (in any order)

## 日本語

剣、ツルハシ、シャベル、斧の機能を併せ持つマルチツール「パクセル」を追加するMod。

### 特徴
- 🔨 どんなブロックも採掘可能
- 💪 通常のツールの4倍の耐久値
- 🪓 斧と同等の攻撃力
- ️⚔️ 剣と同等の攻撃速度と薙ぎ払い攻撃
- (未実装) 右クリック時の挙動

### レシピ
同じ材料の剣・ツルハシ・シャベル・斧をクラフトグリッドに配置(順不同)
